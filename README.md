## 🚀 SpaceX Mission Control
A React dashboard for tracking SpaceX missions and fleet data using the SpaceX API v4.

## 🛠️ Features
Launches: Server-side pagination, status filters, and search (URL persisted).

Ships: Infinite scroll via Intersection Observer and name filtering.

Details: Dynamic routes for specific mission and ship data.

UX/UI: Live launch countdown, responsive layout, and Dark/Light mode (persisted via localStorage).

## 🛰️ Tech Stack
React (Vite), TypeScript, React Router 6.

SpaceX API v4 (POST query endpoints).

CSS Modules with theme-aware variables.

## 🏁 Quick Start
After cloning the repo and navigating to the project root run:

npm install

npm run dev

View the app at: http://localhost:5173
